### Step 1: Data Collection

### Step 2: Data Validation

### Step 3: Normalization
Convert all salaries to:
- Full-time equivalent (FTE)
- Annual base salary (excluding bonuses)
- Australian dollars (AUD)
- Current year values (inflation-adjusted)

### Step 4: Statistical Analysis
Calculate for each role:
- **Median salary** (50th percentile)
- **25th percentile** (entry-level)
- **75th percentile** (experienced)
- **90th percentile** (senior/specialist)
- **Standard deviation**
- **Sample size**

### Step 5: Regional Adjustments
Apply location multipliers:
- **Sydney**: 1.15x base
- **Melbourne**: 1.10x base
- **Brisbane**: 1.00x base (baseline)
- **Perth**: 1.05x base
- **Adelaide**: 0.95x base
- **Canberra**: 1.20x base (public sector premium)
- **Hobart/Darwin**: 0.90x base
- **Regional areas**: 0.85x base

### Step 6: Experience Adjustments
Years of experience multipliers:
- **0-2 years**: 0.75x - 0.90x
- **3-5 years**: 0.90x - 1.10x
- **6-10 years**: 1.10x - 1.30x
- **10+ years**: 1.30x - 1.60x
- **Senior leadership**: 1.60x - 2.50x

## ğŸ’¼ Benefits & Total Compensation

**Base Salary** is just one component. Consider:

### Typical Australian Benefits
- **Superannuation**: 11.5% (mandatory)
- **Annual Leave**: 4 weeks (standard)
- **Sick Leave**: 10 days (standard)
- **Public Holidays**: 10-13 days (state-dependent)

### Additional Benefits (Variable)
- Bonus/Incentive: 10%-30% of base
- Car allowance: $10K-$20K
- Laptop/phone: $2K-$5K
- Professional development: $2K-$10K
- Health insurance: $2K-$5K

### Total Remuneration Formula
```
Total Package = Base Salary + Super + Bonuses + Benefits + Leave Loading
```

**Example:**
- Base: $100,000
- Super (11.5%): $11,500
- Bonus (15%): $15,000
- Car allowance: $15,000
- **Total Package: $141,500**
