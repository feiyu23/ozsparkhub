## ğŸ¥ Workplace First Aid (WFA) Compliance {#wfa-compliance}

### Legal Requirements

**Regulated by**: Work Health and Safety (WHS) Acts (state/territory-specific)

### WFA Compliance Checklist

#### 1. First Aid Assessment âœ…
- [ ] Conduct workplace risk assessment
- [ ] Determine number of first aiders required
- [ ] Identify high-risk work areas
- [ ] Document assessment findings
- [ ] Review assessment annually

#### 2. First Aid Personnel âœ…
- [ ] Appoint trained first aiders
- [ ] Ensure valid certifications (HLTAID011 or equivalent)
- [ ] Maintain certification register
- [ ] Schedule refresher training (every 3 years)
- [ ] Assign deputies for absences

#### 3. First Aid Facilities âœ…
- [ ] Provide adequate first aid kits
- [ ] Check kit contents monthly
- [ ] Ensure kits are clearly signposted
- [ ] Maintain first aid room (if required)
- [ ] Keep emergency contact information accessible

#### 4. Record Keeping âœ…
- [ ] Maintain first aid incident register
- [ ] Record all treatments provided
- [ ] Store records securely (7 years minimum)
- [ ] Report notifiable incidents to regulator
- [ ] Conduct incident reviews

#### 5. Communication âœ…
- [ ] Display first aiders' names and locations
- [ ] Include WFA in induction training
- [ ] Provide multilingual signage (if needed)
- [ ] Update emergency procedures
- [ ] Communicate policy changes

### Minimum First Aider Requirements

| Workplace Size | Low Risk | High Risk |
|----------------|----------|-----------|
| 0-25 workers | 1 first aider | 2 first aiders |
| 26-100 workers | 1 per 50 workers | 1 per 25 workers |
| 100+ workers | Additional as required | Additional as required |

**High-risk industries**: Construction, manufacturing, healthcare, mining

### DEI Strategy Checklist

#### 1. DEI Assessment âœ…
- [ ] Conduct workforce demographic analysis
- [ ] Survey employee perceptions
- [ ] Identify representation gaps
- [ ] Review policies for bias
- [ ] Benchmark against competitors

#### 2. Policy Development âœ…
- [ ] Create DEI policy statement
- [ ] Develop anti-discrimination policy
- [ ] Establish flexible work policy
- [ ] Implement inclusive language guidelines
- [ ] Draft reasonable accommodation procedures

#### 3. Recruitment DEI âœ…
- [ ] Use diverse interview panels
- [ ] Implement structured interviews
- [ ] Expand talent sourcing channels
- [ ] Partner with diversity organizations
- [ ] Monitor diversity metrics in hiring

#### 4. Workplace Culture âœ…
- [ ] Mandatory unconscious bias training
- [ ] Create employee resource groups (ERGs)
- [ ] Celebrate cultural diversity events
- [ ] Provide inclusive facilities (prayer rooms, gender-neutral bathrooms)
- [ ] Use inclusive language (pronouns, name pronunciation)

#### 5. Career Development âœ…
- [ ] Ensure equitable promotion practices
- [ ] Provide sponsorship programs
- [ ] Offer leadership training for underrepresented groups
- [ ] Conduct pay equity audits
- [ ] Create succession planning for diversity

#### 6. Measurement & Accountability âœ…
- [ ] Track diversity metrics (gender, ethnicity, disability, age)

| Metric | Target Example | Frequency |
|--------|----------------|-----------|

## ğŸ“œ General Employment Compliance {#general-compliance}

### Essential Compliance Areas

#### 1. Employment Contracts âœ…
- [ ] Written employment contracts for all employees
- [ ] Include all legally required terms
- [ ] Specify award/agreement coverage
- [ ] Clearly state employment type (full-time/part-time/casual)
- [ ] Define probation period (typically 3-6 months)

#### 2. Fair Work Compliance âœ…
- [ ] Pay at least minimum wage
- [ ] Apply correct modern award
- [ ] Pay penalty rates correctly
- [ ] Provide required leave entitlements
- [ ] Issue pay slips within 1 working day
- [ ] Keep employment records for 7 years

#### 3. Superannuation âœ…
- [ ] Pay super to eligible employees (earning $450+/month)
- [ ] Pay 11.5% of ordinary time earnings (as of July 2025)
- [ ] Pay super quarterly (or monthly if in pay)
- [ ] Use employee's chosen super fund
- [ ] Report super contributions via Single Touch Payroll (STP)

#### 4. Tax Compliance âœ…
- [ ] Obtain TFN declarations
- [ ] Register for PAYG withholding
- [ ] Calculate tax correctly
- [ ] Report via Single Touch Payroll (STP)
- [ ] Issue payment summaries (if required)
- [ ] Provide FBT information

#### 5. Work Health & Safety âœ…
- [ ] Provide safe work environment
- [ ] Conduct risk assessments
- [ ] Provide necessary PPE
- [ ] Offer WHS training
- [ ] Report notifiable incidents
- [ ] Maintain incident register

#### 6. Record Keeping âœ…
- [ ] Employment records (7 years)
- [ ] Time and wages records
- [ ] Leave records
- [ ] Super contributions
- [ ] Tax withholding records
- [ ] Training certificates

#### 7. Termination Compliance âœ…
- [ ] Provide minimum notice period
- [ ] Pay all outstanding entitlements
- [ ] Issue separation certificate
- [ ] Conduct exit interview
- [ ] Final super payment
- [ ] Return of company property
