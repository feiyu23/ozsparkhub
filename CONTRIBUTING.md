## ğŸ› Reporting Issues

### Before Reporting
- Check if the issue already exists
- Try reproducing the issue
- Gather relevant details (browser, device, etc.)

### How to Report

**Option 1: GitHub Issues** (preferred)
- Use the issue template
- Provide clear, detailed description
- Include screenshots if applicable
- Add steps to reproduce

**Option 2: Email**
- Send to: hello@ozsparkhub.com.au
- Subject: "Bug Report: [Brief Description]"
- Include all relevant details

### What We Need
- **Clear Title**: Describe the issue concisely
- **Description**: What happened vs. what you expected
- **Steps to Reproduce**: How to trigger the issue
- **Impact**: How serious is the issue?
- **Environment**: Browser, device, OS version
- **Screenshots**: Visual evidence helps!
